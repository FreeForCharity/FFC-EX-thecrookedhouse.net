# Contributing to The Crooked House Website

Thank you for your interest in contributing to The Crooked House website!

## Repository

This repository is hosted at: [https://github.com/FreeForCharity/FFC-EX-thecrookedhouse.net](https://github.com/FreeForCharity/FFC-EX-thecrookedhouse.net)

## How to Contribute

### Reporting Issues

If you find any issues with the website, please open an issue in the GitHub repository.

### Making Changes

1. Fork the repository from [https://github.com/FreeForCharity/FFC-EX-thecrookedhouse.net](https://github.com/FreeForCharity/FFC-EX-thecrookedhouse.net)
2. Create a new branch for your changes
3. Make your changes to the HTML files
4. Test your changes locally by opening the HTML files in a web browser
5. Commit your changes with a clear commit message
6. Push your changes to your fork
7. Open a pull request to the main repository

### Testing

Since this is a static HTML website, you can test changes by:
- Opening the HTML files directly in your web browser
- Using a local web server (e.g., `python -m http.server` or `npx serve`)

### Deployment

Changes merged to the `main` branch are automatically deployed to GitHub Pages via GitHub Actions.

## Questions?

If you have any questions, please open an issue in the repository or contact the project maintainers.
